Cultura MX.
Es un proyecto que consiste en:
La creacion de un sitio web dedicado a contener informacion reelevante de las culturas de mexico,
ocpuando: HTML, CSS Y Javascript.
Consta de varias etapas.

__________________________________
Etapa 1: LA BASE PARA EL SITIO WEB: Cultura Antigua.

Etapa 1.1: Esqueleto.
Sacar toda la informacion de una sola cultura a elegir (La mas pupular).
Detallarla lo mas que se pueda en un documento .docx

Etapa 1.2: Crear el formato HTML.
Dar la estructura HTML e introduccir la informacion de documento.

Etapa 1.3: Proceso creativo.
Aplicar CSS a la estrucutra HTML, 
basandonos en paginas web que tengan el mismo giro, 
es decir, paginas que son similares a la del proyecto.


_______________________________________
Etapa 2: SITIO WEB (HTML CSS JAVASCRIPT)
Una vez terminada la etapa 1.
Se le pasara a otras paginas el script. 
Para que todas tengan la misma estructura del script.
